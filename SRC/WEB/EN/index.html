Top 2000 App for MS-DOS

With the Top 2000 app you have a complete overview of all the tracks and all editions, since 1999, in one simple application.

The most user friendly app for the Top 2000 specially written for Microsoft DOS. The app is designed for a fluent experience and an easy way to keep track of the list.

The Top 2000 is an annual radio program from the Dutch Radio agency NPO 2. It starts at midnight on the first day of Chirstmas and runs 24/7 till new year's eve.
