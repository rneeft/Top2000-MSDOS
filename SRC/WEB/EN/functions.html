Functions

The main screen

On the main screen a total of 10 tracks are being shown at once. For each track the position, of the given edition, the title and the artist is displayed.

Navigation

With the menu different options are displayed for easy navigation through the list.

Next to those options it is possible to navigation by using the scroll bar.

Detail screen

It is possible to find more details by clicking on a track, the detail screen will then be shown.

On the detail screen more statistic detail is displayed for the given track. You will find the title, the artist and the on which year the track has been made. All the appearances, since the first edition in 1999, are listed horizontally. Each icon idicates whether the track has rise, dropped, came back, was new or wasn't displaed at all.

Tot total amount of appearances are shown in two different ways. The first type displayed how many times the track was listed compared to how many times it could have been listed. For example, Oceaan from Racoon was release in 2013 and have been listed ever since. The second type is how many times the track was listed since the start of the TOP2000.

In the last section is displayed the highest/lowest and the first/last appearance of the track. When the track is listed in the last edition then also the date and time is shown in which the track can be heared (CET-time).

Switching of edition

In the menu 'Editie' (or with CTRL-S) it is possible to change the edition. Type the year of the edition in the textbox or use the arrow up and down to switch.
