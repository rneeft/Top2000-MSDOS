Sourcecode

The Top2000 is developed in Visul Basic 1.0 for DOS. The sourcode is open source and can be found here. Use the following code to clone the souce with Git:

git clone https://github.com/rneeft/Top2000-MSDOS.git

Application

On modern systems the Top2000 voor DOS can be run inside a DOS emulator. When you make use of the installer Top2000Installatie.exe, the TOP2000 will be installed with DOS emulator DOSBox. The app can be opened with a shortcut on the startmenu.

When you have started the TOP2000 app for DOS it is possible to make the application full screen by pressing CTRL-ENTER. Press it again to make it normal again.

Real hardware

Off course it is possible to run the application on real hardware. Make sure Microsoft DOS 5.0 or higher is running on the system. You have two options for installation:

  • Top2000 floppy image.
    Download this option to copy it onto a 1.44MB floppy disk. Read the Leesmij (Dutch for Readme) file for installation options.
  • Top2000 ZIP.
    Choose this option when there is another way to move the files to the system.
