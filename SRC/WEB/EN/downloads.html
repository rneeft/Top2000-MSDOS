Source code

The Top2000 is developed in Visul Basic 1.0 for DOS. The source code is open source and can be found here. Use the following code to clone the souce with Git:

git clone https://github.com/rneeft/Top2000-MSDOS.git

Application

There are several ways to enjoy the TOP2000 app for MS-DOS. If you want to run the application on a modern Windows/Mac or Linux system, please make use of DOSBox

  • TOP2000 application including DOSBox
    With this version you can quickly start using the TOP2000 application with DOSBox. Run the application by executing the TOP2000.BAT file.

  • Top2000 application for a 1.44 MB floppy
    Download this zip file when you want to install the Top2000 application on real hardware. You need Microsoft MS-DOS 5.0 or higher in order to run the software. Read the (Dutch) leesmij.txt file for more installation instructions.

  • Top2000 Application
    This file only contains the Top2000 app, icon and the data file.

Windows security

When you download the software you might need to unblock the ZIP file. Right click the file and click on properties. Unlock the file by checking the 'Unblock' checkbox. Click on OK and extract the ZIP file.
