Broncode

Top2000 is ontwikkeld in Visual Basic 1.0 for DOS. De code is open source en kan hier bekeken worden. U kunt de code makkelijk clonen met Git door middel van het command:

git clone https://github.com/rneeft/Top2000-MSDOS.git

Applicatie

Op moderne systemen kan de TOP2000 voor DOS gedraaid worden in een DOS emulator. Als u gebruikt maakt van het installatiebestand Top2000Installatie.exe wordt de TOP2000 voor u geinstalleerd met een DOS emulator. Via een snelkoppeling vanaf uw startmenu start u gemakkelijk de applicatie op.

Wanneer u de TOP2000 app voor DOS hebt gestart kunt u door middel van CTRL-ENTER de applicatie in volledig scherm draaien. Klik op uw windows toets om het volledig scherm weer af te sluiten.

Echte hardware

Natuurlijk kunt u de TOP2000 ook installeren op echte hardware. Hiervoor heeft u minimaal Microsoft MS-DOS 5.0 nodig. U heeft keuze uit twee bestanden.

  • Top2000 diskette image.
    Kies deze optie wanneer u de TOP2000 op een 1.44 MB diskette wilt plaatsen. Bekijk het Leesmij bestand voor installatie instructies.
  • Top2000 ZIP.
    Kies deze optie wanneer u op een andere manier de TOP2000 app voor DOS wilt bekijken.
