Broncode

Top2000 is ontwikkeld in Visual Basic 1.0 for DOS. De code is open source en kan hier bekeken worden. U kunt de code makkelijk clonen met Git door middel van het command:

git clone https://github.com/rneeft/Top2000-MSDOS.git

Applicatie

Indien u gebruik wilt maken van echte hardware dan kunt u een image downloaden voor een diskette. Op het image staat een leesmij.txt bestand met instructies hoe u de applicatie kunt installeren en eventueel bewerken. De tool vereist minimaal Microsoft DOS 5.0

Top2000 diskette image

U kunt ook gebruik maken van MS-DOS emulatie software. Een gratis DOS emulator is DOSBox. Met DOSBox kunt u gemakkelijk oude DOS applicaties gebruiken. DOSBox kunt hier downloaden. Voor DOSBox kunt u gebruik maken van het TOP2000 ZIP bestand.

Top2000 ZIP

Installatie met DOSBox

U kunt de TOP2000 app voor DOS in DOSBox gebruiken, doorloop daarbij de volgende stappen of kijk mee met onderstaande video.

  1. Download en installeer DOSBox op uw computer
  2. Download de Top2000 ZIP hier
  3. Pak de content
