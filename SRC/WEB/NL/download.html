Broncode

Top2000 is ontwikkeld in Visual Basic 1.0 for DOS. De code is open source en kan hier bekeken worden. U kunt de code makkelijk clonen met Git door middel van het command:

git clone https://github.com/rneeft/Top2000-MSDOS.git

Applicatie

Er zijn verschillende manier om van de TOP2000 voor DOS te genieten. Maak gebruik van DOSBox om de applicatie te runnen op een modern Windows/Mac/Linux systeem.

  • TOP2000 applicatie inclusief DOSBox
    Deze versie bevat de TOP2000 applicatie met DOSBox zodat u snel aan de slag kunt. Bekijk het Leesmij bestand met meer informatie hoe u de applicatie kunt starten.

  • Top2000 applicatie voor 1.44 MB Diskette
    Kies deze variant wanneer u de Top2000 app wilt installeren op echte hardware. U heeft minimaal Microsoft DOS 5.0 nodig om de software goed te kunnen draaien. Lees het leesmij bestand voor meer installatie instructies.

  • Top2000 applicatie
    Deze release bevat alleen de Top2000 app, icoon en het data bestand.

Windows beveiliging

Wanneer u de software heef gedownload dient u eerst het ZIP bestand te deblokkeren. Klik met uw rechter muisknop op het ZIP bestand en kies voor Eigenschappen. Deblokkeer het bestand vervolgens door onderaan een vinkje te zetten bij 'Blokkering opheffen'. Druk daarna op OK en pak het ZIP bestand uit.
